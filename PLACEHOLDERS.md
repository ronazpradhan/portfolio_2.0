# 🔧 Placeholders to Fill In

Search for `YOUR_` in the codebase and replace:

| Placeholder | File | What to replace with |
|---|---|---|
| `YOUR_GITHUB_URL` | Hero.tsx, Projects.tsx, Contact.tsx | Your GitHub profile URL |
| `YOUR_LIVE_URL` | Projects.tsx (Project 1) | Live URL of your portfolio |
| `YOUR_LINKEDIN_URL` | Contact.tsx | Your LinkedIn profile URL |
| `your@email.com` | Contact.tsx | Your actual email |
| `github.com/yourusername` | Contact.tsx | Your GitHub handle |
| `linkedin.com/in/yourusername` | Contact.tsx | Your LinkedIn handle |

## Project GitHub links
Each project in `Projects.tsx` has its own `github` field — fill in per project.
